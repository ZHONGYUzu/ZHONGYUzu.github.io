Safety Gear Detection System

Automated compliance monitoring using YOLOv8 object detection.

üõë The Challenge

Construction sites face significant safety risks when workers neglect to wear Personal Protective Equipment (PPE) like helmets and vests. Manual monitoring by supervisors is intermittent and error-prone.

Goal: Build a real-time computer vision system that automatically flags workers without safety gear in video feeds.

‚ö° The Solution

I implemented the YOLO (You Only Look Once) algorithm. Unlike older methods (like R-CNN) that scan images multiple times, YOLO looks at the entire image once, making it fast enough for live video (45+ FPS).

üõ†Ô∏è Implementation Details

I used the Ultralytics YOLOv8 library. I fine-tuned a pre-trained model on a dataset of 5,000 construction site images.

Inference Code Snippet

from ultralytics import YOLO
import cv2

# 1. Load the custom trained model
model = YOLO('best_weights.pt')

# 2. Run inference on a video source
results = model.predict(source='site_video.mp4', conf=0.5)

# 3. Process results
for result in results:
    boxes = result.boxes
    print(f"Detected {len(boxes)} objects")
    result.save() # Save the frame with bounding boxes

üìä Performance Metrics

The model was evaluated on a held-out test set. We prioritized Recall (finding all violations) over Precision.

  • mAP@0.5: 0.94 (High accuracy)
  • Inference Speed: 12ms per frame (Real-time on Tesla T4 GPU)
  • Classes: Helmet, Vest, No-Helmet, No-Vest
